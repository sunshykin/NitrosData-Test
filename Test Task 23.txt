Тестовая задача № 23

Написать Web приложение ведения списка персон, среди которых могут быть родственники. 

Приложение содержит 2 формы:
- Master
- Detail

Форма Master содержит 
- Таблицу (Grid) персон следующей структуры
  * Фамилия
  * Имя
  * Отчество
  * Дата рождения
  * Адрес
- Поля поиска по следующим данным:
  * Фамилия
  * Имя
  * Дата рождения
- Кнопки 
  * Insert
  * Delete
  * Edit

Кнопка Delete запрашивает подтверждение удаления, удаляет в случае подтверждения. 

Кнопка Edit вызывает редактирование детализированных данных (Фамилия, Имя, Отчество, Дата рождения, Адрес). 

Кнопка Insert дает возможность ввода детализированных данных по новой персоне. 

На форме Detail отображается таблица родственников с возможностью выбора (добавления, удаления) родственников для данной персоны. 
На данной форме нет необходимости заводить новую персону, здесь достаточно выбрать родственника из персон, уже заведенных ранее в базе.
